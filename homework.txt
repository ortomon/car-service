Создать сервис объявлений машин (все действия в сервисе ниже являются http endpoint).

1. Сервис позволяет создать объявление (уникальный идентификатор (генерите внутри java), имя бренда, цвет,
цена, модель, дата размещения)

2. Сервис позволяет получить все объявления за дату

3. Сервис позволяет удалить объявление (по уникальному идентификатору)

4. Сервис позволяет найти машины по параметрам (если указано одно поле, то поиск по одному полю.
Если указано два и более то сочетание этих полей): имя бренда, цвет, цена, модель

5. Сервис позволяет найти машину по уникальному идентификатору.

6. Расположить приложение на 8003 порту.
